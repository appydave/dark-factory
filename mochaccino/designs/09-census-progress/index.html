Self-Improvement Grounding
SkillOpt — Microsoft Research

Core principle: The skill file is the external trainable state of a frozen agent. The agent model never changes; only the skill file does. Optimization is text-space: bounded add/delete/replace edits, accepted only when they strictly improve a held-out validation score.

Status in this system: The design is implemented. The code is not yet built. Level 2b (rollout evaluation) is the first place SkillOpt's mechanism applies — run the skill on held-out tasks, score the outputs, accept edits that improve the score. The Dark Factory running its own management workflows generates real training signal: every Level 1 census run that David reviews is a scored rollout.
